etiqueta <"html">



Tipo de etiqueta: Etiqueta contenedora (tiene contenido)

Propósito: Definir el documento HTML y contener todas las demás etiquetas.

Atributos:
- lang: especifica el idioma del documento

- xmlns: especifica el namespace XML (opcional)

Estructura:

html

/html

Importancia:

1. Define el documento HTML
2. Contiene las etiquetas head y body
3. Es el elemento principal del documento

Consecuencias de no utilizarla:

1. El documento no es válido
2. Los navegadores pueden tener problemas para renderizar el contenido
3. La estructura del documento no es clara

Recomendación:

- Utilizar siempre la etiqueta html como etiqueta raíz
- Especificar el idioma con el atributo lang
- No omitir la etiqueta html en documentos HTML
